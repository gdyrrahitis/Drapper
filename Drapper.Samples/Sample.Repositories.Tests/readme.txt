-----------------------------------------------------------------------------------------------
BEFORE YOU BEGIN:
-----------------------------------------------------------------------------------------------
• Create the DrapperSamples database using the sql file in .\Sql\CreateDatabase.sql.
• Update the connection string in Drapper.Settings.json to point at the sample database.

-----------------------------------------------------------------------------------------------
WHAT IS THIS?
-----------------------------------------------------------------------------------------------
This is a small sample project to highlight a few of the features of Drapper to 
get you started. The sample is a little contrived, with the focus being on how 
Drapper can make your day to day coding easier.

The test project doesn't contain any unit tests. It contains integration tests 
against the samples database to give you examples of what working with Drapper 
is like.
